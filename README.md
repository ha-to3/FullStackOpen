# [University of Helsinki - Full Stack Open 2022](https://fullstackopen.com/en/)

This course introduce to modern JavaScript-based web development. The main focus is on building single page applications with ReactJS that use REST APIs built with Node.js. This course covers: React, Redux, Node.js, MongoDB, GraphQL and TypeScript.
